What Nullcore does

ECS

Entity Component System

Powered by EnTT. Every scene object is an entity. Components: Transform, MeshRenderer, Camera, Light, NativeScript. RenderSystem and ScriptSystem iterate them automatically each frame.

Renderer

OpenGL 4.6 Core Profile

No legacy fixed-function pipeline. Abstracted RendererAPI, VertexArray, Buffer, Shader with uniform helpers, Texture2D via stb_image. Both perspective and orthographic cameras.

Scene

Scene Management

Runtime and editor update loops. RenderSystem submits all visible MeshRenderers, ScriptSystem ticks all NativeScriptComponents. SceneSerializer handles save and load.

Scripting

Native C++ Scripts

Derive from ScriptableEntity, implement OnCreate / OnUpdate / OnDestroy, bind with one line. Runs directly inside the ECS loop — zero overhead, no VM, no reflection.

Camera

3 Camera Modes

Orbit arc-ball, Free Noclip (WASD + RMB look, Shift sprint, F1 mouse lock like CS/MC), and per-cube rotation. Switch at runtime with 1/2/3 — camera state transfers seamlessly.

Events

Typed Event System

LayerStack dispatches typed events via templated callbacks. KeyPressed, MouseMoved, MouseButton, MouseScrolled, WindowResize. No string matching, no dynamic_cast spam.

Engine Architecture

Project Layout

NullcoreEngine        ← static .lib
├── Core/
│   ├── Application    ← main loop
│   ├── LayerStack     ← ordered layers
│   ├── Log            ← debug output
│   └── UUID           ← entity IDs
│
├── Renderer/
│   ├── RendererAPI    ← abstract GL
│   ├── Renderer       ← Begin/Submit/End
│   ├── Buffer         ← VBO / IBO
│   ├── VertexArray    ← VAO
│   ├── Shader         ← GLSL + uniforms
│   ├── Texture2D      ← stb_image
│   └── Camera         ← Ortho + Persp
│
├── Scene/ (ECS via EnTT)
│   ├── Scene          ← registry + systems
│   ├── Entity         ← handle wrapper
│   ├── Components     ← all structs
│   ├── ScriptableEntity
│   └── SceneSerializer
│
└── Platform/Windows/
    ├── WindowsWindow  ← GLFW + events
    ├── OpenGLBuffer
    ├── OpenGLVertexArray
    └── OpenGLTexture

Sandbox                ← exe, links .lib
└── SandboxApp     ← ECS demo

ECS — Example Usage

// create entity in scene
Entity e = scene->CreateEntity("Cube");

// transform
auto& tc = e.GetComponent
    <TransformComponent>();
tc.Translation = { 0, 0, 0 };
tc.Scale       = { 1, 2, 1 };

// mesh renderer
auto& mesh = e.AddComponent
    <MeshRendererComponent>();
mesh.MeshShader = shader;
mesh.VAO        = vao;
mesh.Albedo     = texture;

// native C++ script
e.AddComponent<NativeScriptComponent>()
 .Bind<RotatorScript>();

// one call runs everything
scene->OnUpdateRuntime(dt);
// RenderSystem → draws meshes
// ScriptSystem → ticks scripts

Tech Stack

01C++20 / MSVCModern language features — templates, structured bindings, concepts. Built with Visual Studio 2026.Core
02OpenGL 4.6 + GLADCore profile only. GLAD for extension loading. No deprecated fixed-function pipeline.Graphics
03GLFWWindow creation, OpenGL context, keyboard and mouse input. Wrapped behind the engine's interfaces.Platform
04GLMvec2/3/4, mat4, quat, perspective, lookAt. Used in every component, system, and camera.Math
05EnTTHigh-performance ECS registry. Sparse-set storage, compile-time type IDs, clean view API.ECS
06stb_imageSingle-header image loader. PNG, JPEG, BMP. Wrapped inside Texture2D::Create().Assets
// 04 — getting started

Build from Source

01

Clone the repository

All dependencies are vendored — no package manager needed.

git clone https://github.com/onemorefix1337/nullcore.git
02

Open in Visual Studio 2026

Open NullcoreEngine.slnx. Two projects: NullcoreEngine (static lib) and Sandbox (executable).

03

Set Release x64

Select Release and x64 in the toolbar. Debug also works.

04

Build and run

Ctrl+Shift+B to build, F5 to run. Sandbox opens with 3 rotating cubes and all camera modes.

// 05 — sandbox

Sandbox Controls

1Cube mode — LMB drag rotates selected cube
2Orbit mode — LMB orbits camera, Scroll zooms
3Free Cam — noclip flight mode
TabCycle selected cube (highlighted yellow) in Cube mode
WASDMove camera in Free Cam
Q / EMove down / up (world space, not camera-relative)
ShiftSprint — 4× movement speed
RMBHold to look around in Free Cam
F1Toggle mouse lock — cursor hidden and captured (CS / MC style)
ScrollZoom in / out in Orbit mode
EscQuit application
